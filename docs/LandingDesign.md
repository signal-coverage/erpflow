# Premium Logistics Landing Page Prompt

Create a modern, premium logistics company landing page inspired by high-end SaaS and transportation websites. The design should feel clean, corporate, trustworthy, and visually impressive, using large photography, rounded cards, plenty of whitespace, and subtle glassmorphism effects.

---

# Design Style

- Modern minimalism
- Premium enterprise branding
- White background
- Orange (`#FF7A1A`) as the primary accent color
- Dark gray typography
- Rounded corners (20–32px)
- Soft shadows
- Glassmorphism navigation
- Large imagery
- Smooth scrolling sections
- Elegant spacing
- Responsive design
- Built with React + Tailwind CSS

- Font: **Manrope** or **Plus Jakarta Sans**

Use high-quality professional photography featuring:

- Container ships

---

# Overall Goal

The final landing page should feel like a **premium enterprise logistics website**, combining the elegance of a modern SaaS landing page with the credibility of a global shipping company.

Emphasize:

- Large immersive visuals
- Rounded layouts
- Smooth animations
- Glassmorphism elements
- Strong typography
- Clear calls to action
- Excellent visual hierarchy
- Mobile-first responsive behavior
- Accessibility best practices
- Fast loading performance

The result should look polished enough to be used by a global logistics brand while maintaining the modern aesthetic commonly seen in award-winning web designs.
